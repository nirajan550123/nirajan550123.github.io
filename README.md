# nirajan550123.github.io

Personal portfolio of **Nirajan Tripathi** — GIS Analyst and geospatial data
scientist. M.S. Geography candidate at Texas State University, with a background
in geomatics engineering, remote sensing, and spatial data science.

**Live site:** https://nirajan550123.github.io/

## About

The site presents reproducible geospatial work across environment, water
resources, and urban planning — from satellite remote sensing and spatial
modeling to web mapping. Each project links to its own code and documentation.

Built as a static site with vanilla HTML, CSS, and JavaScript, hosted on GitHub
Pages. No framework or build step.

## Projects

| Project | Focus | Tools |
|---------|-------|-------|
| [Austin Urban Heat Island Story Map](./projects/austin-heat-storymap/) | Interactive scrollytelling map of summer land surface temperature across 65 neighborhoods, with a PostGIS census-income spatial join | Earth Engine, PostGIS, R, Leaflet, Chart.js |
| [Rooftop Solar Across Data Environments](./projects/lidar-rooftop-solar/) | Same rooftop-solar method on 1 m LiDAR and 30 m DSM, quantifying resolution bias | ArcGIS Pro, USGS 3DEP, Copernicus GLO-30, Python |
| [Watershed Land Cover Simulation](./projects/watershed-lulc-markov/) | Markov chain–Cellular Automata model forecasting land-cover change | ArcGIS Pro, Markov-CA, Python, NLCD |
| [Transit Dependency Likelihood Index](./projects/tdli-portland/) | Custom ArcPy tool combining demographic need with transit service areas | ArcPy, Network Analyst |
| [NDVI, LST & Land Cover](./projects/ndvi-lst-hillcountry/) | Vegetation cooling across land-cover types, reproducible R workflow | ArcGIS Pro, Landsat, R |
| [Post-Fire Vegetation Damage Assessment](./projects/palisades-fire-rs/) | Pre/post-fire change detection using hyperspectral imagery | ERDAS, hyperspectral, red-edge indices |
